~Running MCSH~

1) The included makefile creates a 'prog4_dam0271' executable to run MCSH
2) After 'make' command './prog4_dam0271' will start MCSH
	** Note: './prog4_dam0271 test.txt' will run MCSH with included test file **
3) The 'make clean' command will remove all *.o's and prog4_dam0271
